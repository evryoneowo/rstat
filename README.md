# rstat
light stat utility

## installation
1. `git clone https://github.com/evryoneowo/rstat && cd rstat`
2. `make`
3. `sudo make install`

## how to use
1. `rstat` - usual mode
2. `rstat -l` - monitor mode
3. `rstat -v` - version & info
4. `rstat -n {num}` - specify the num of processes to show
